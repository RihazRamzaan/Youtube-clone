# YouTube Frontend Clone 

A frontend clone of YouTube’s user interface, built with HTML and CSS as a hands-on project to enhance frontend development skills and practice GitHub workflows.
---

## Project Objective
This project aims to replicate YouTube’s UI using only HTML and CSS, focusing on clean code structure, responsive design, and effective use of Git for version control.

---

## Key Features

- Responsive Header: Includes a logo, search bar, and user profile section.
- Sidebar Navigation: Mimics YouTube’s menu for easy navigation.
- Video Grid Layout: Displays video thumbnails with titles and channel information.
- Clean CSS Styling: Organized and maintainable styling for a polished look.
- Basic Responsive Design: Ensures the layout adapts to different screen sizes.


---
